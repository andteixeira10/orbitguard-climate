3. DESCRIÇÃO DA SOLUÇÃO

O OrbitGuard Climate é uma API conteinerizada para monitoramento de áreas de risco climático usando dados simulados de satélite.

A solução conecta o tema da economia espacial com um problema real da Terra: o acompanhamento de áreas vulneráveis a eventos climáticos como seca, calor extremo, baixa umidade e ausência de chuva.

A aplicação permite cadastrar áreas monitoradas, registrar leituras orbitais simuladas e gerar alertas climáticos relacionados aos dados coletados.

A solução foi desenvolvida com .NET 9 Web API, MySQL 8.0, Entity Framework Core, Docker, Docker Compose e Swagger.

6. MODELO DE DADOS

A solução possui três tabelas principais:

6.1 monitored_areas

Representa uma área monitorada.

Exemplo:
Fazenda Aurora - Ribeirão Preto/SP.

6.2 satellite_readings

Representa uma leitura simulada de satélite associada a uma área monitorada.

Exemplo:
Temperatura, umidade, chuva e índice de vegetação.

6.3 climate_alerts

Representa um alerta climático associado a uma área e, opcionalmente, a uma leitura de satélite.

Exemplo:
DROUGHT_RISK, severidade CRITICAL, status IN_ANALYSIS.


7. RELACIONAMENTO ENTRE TABELAS

monitored_areas 1:N satellite_readings

monitored_areas 1:N climate_alerts

satellite_readings 1:N climate_alerts

Esse relacionamento foi validado por SELECT com JOIN diretamente dentro do container MySQL.
